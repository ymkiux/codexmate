# Getting Started

## Install

```bash
npm install -g codexmate
```

Or run without global install:

```bash
npx codexmate@latest status
```

## Basic Commands

```bash
codexmate setup
codexmate status
codexmate run
```

- `setup`: interactive initialization.
- `status`: check current provider/model/profile state.
- `run`: open local Web UI.
